# GlobalCrush.pk

Celebrity Lifestyle & Fashion Website - 100% Free Setup

## Features

- Modern, responsive design
- Mobile-first approach
- SEO optimized
- Ad-ready (PopAds, HilltopAds, Adsterra)
- Fast loading
- Static site (no server needed)

## Deploy on Vercel (Free)

1. Push this folder to GitHub
2. Go to vercel.com
3. Import your GitHub repository
4. Click Deploy
5. Done! Your site is live

## Deploy on Netlify (Free)

1. Push this folder to GitHub
2. Go to netlify.com
3. Import your GitHub repository
4. Click Deploy
5. Done!

## Deploy on Render (Free)

1. Push this folder to GitHub
2. Go to render.com
3. Create new Static Site
4. Connect your GitHub repo
5. Set build command: (leave empty)
6. Set publish directory: .
7. Click Create

## Ad Integration

### PopAds (Popunder)
1. Sign up at popads.net
2. Add your website
3. Generate popunder code
4. Replace `<!-- PLACEHOLDER: PopAds Popunder Code -->` in index.html

### HilltopAds (Banners + MultiTag)
1. Sign up at hilltopads.com
2. Add your website (select Non-Mainstream)
3. Create ad zones (728x90, 300x250, MultiTag)
4. Replace placeholder comments in HTML files

### Adsterra (Social Bar + Smartlink)
1. Sign up at adsterra.com
2. Add your website
3. Generate Social Bar + Smartlink codes
4. Replace placeholder comments in HTML files

## Content Management

- Edit `data/articles.json` to add/modify articles
- Images from Unsplash (free to use)
- Categories: Fashion, Beauty, Lifestyle, Celebrity, Style, Travel
